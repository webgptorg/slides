# Promptbook — AI Agents as Books

Investor Pitch Deck

• Up-to-date build: 2026-03-20

---

## One-line pitch

Promptbook turns company knowledge, rules and personality into portable, auditable "Books" that run reliable, controllable AI agents across chat, email, code, and internal systems.

---

## Problem

• Companies struggle to convert scattered documents, policies and tribal knowledge into consistent, auditable AI behaviour.

• Prompt drift, unpredictable model outputs and hard-to-maintain prompt engineering create risk for business adoption.

• Enterprises need portable, testable, and enforceable AI agents with fine-grained rules and governance.

---

## Solution

Promptbook introduces Book language — a human-friendly domain language to declare Personas, Knowledge, Rules and Actions. The Promptbook Engine and Server run, enforce, and deploy these Books as production AI agents with RAG, rule enforcement and integrations.

Sources: ptbk.io, Promptbook GitHub

---

## Product & Key Features

• Book language: write explicit commitments (PERSONA, KNOWLEDGE, RULE, ACTION) to define behavior.

• Promptbook Engine: converts Books into model parameters, retrieval flows and enforcement techniques.

• Agent Server (Docker): manage, deploy and host agents.

• Integrations: OpenAI, Anthropic, Gemini, Ollama, Azure, Google, PDF/DOC ingestion, website crawling and vector RAG.

• Pricing & deployment options: Free, Pro, Enterprise; on-prem and managed offerings for municipalities and large orgs.

Sources: ptbk.io, Promptbook GitHub
